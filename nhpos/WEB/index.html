Welcome to the NeighborhoodPOS Web Server

This web server is a small web server built into the NeighborhoodPOS terminal to provide a platform allowing Third Party Vendors to develop applications that use the web services supplied by the NeighborhoodPOS Web Server. The web services provide a mechanism to access the data and the provisioning information of the NeighborhoodPOS terminal.

The NeighborhoodPOS Web Server is designed to allow the use of most modern web interfaces including the use of CSS stylesheets, Javascript files, and XML. The web server has a set of web services that it offers. The number and types of services offered will expand in the near future.

Current Web Services Offerings

    Status Reports

  • /systemstatus.sys -> returns the system status as a text file similar to the AC888 report

    AC21 Totals Reports

  • /databasequery.sys?ac21dailycur -> returns the AC21 Current Daily report
  • /databasequery.sys?ac21dailysav -> returns the AC21 Saved Daily report
  • /databasequery.sys?ac21ptdcur -> returns the AC21 Current Period to Date report
  • /databasequery.sys?ac21ptdsav -> returns the AC21 Saved Period to Date report

    AC23 Totals Reports

  • /databasequery.sys?ac23dailycur -> returns the AC23 Current Daily report
  • /databasequery.sys?ac23dailysav -> returns the AC23 Saved Daily report
  • /databasequery.sys?ac23ptdcur -> returns the AC23 Current Period to Date report
  • /databasequery.sys?ac23ptdsav -> returns the AC23 Saved Period to Date report
